# 📚 DSA Learning Roadmap

A comprehensive, interactive website for learning Data Structures & Algorithms (DSA) with a structured day-by-day roadmap. Perfect for interview preparation, competitive programming, or mastering computer science fundamentals.

## 🎯 Features

- **8-Week Structured Learning Path** (56 days)
- **Daily Topics & Tasks**: Organized topics for each day with difficulty levels
- **Progress Tracking**: Mark topics as completed and track your progress locally
- **Interactive UI**: Beautiful, responsive design with smooth animations
- **Resource Links**: Direct links to YouTube tutorials for each topic
- **Week Filtering**: View all weeks or focus on specific weeks
- **Persistent Storage**: Your progress is saved in your browser
- **Mobile Friendly**: Fully responsive design works on all devices

## 📖 Roadmap Overview

### Week 1: Foundations
- Big-O Notation & Complexity Analysis
- Arrays & Strings
- Linked Lists
- Stack & Queue

### Week 2: Advanced Data Structures
- Trees & Binary Search Trees
- AVL Trees & Heaps
- Hash Tables
- Graphs

### Week 3: Searching & Sorting
- Binary Search & Linear Search
- Sorting Algorithms (Bubble, Selection, Insertion, Merge, Quick, Heap, etc.)
- Sorting Applications

### Week 4: Graph Algorithms I
- DFS & BFS
- Connected Components
- Topological Sorting
- Shortest Path Algorithms (Dijkstra, Bellman-Ford)

### Week 5: Graph Algorithms II & DP Intro
- Floyd-Warshall Algorithm
- Minimum Spanning Tree (Kruskal's, Prim's)
- Dynamic Programming Basics

### Week 6: Dynamic Programming Advanced
- Knapsack Problems
- Longest Common Subsequence
- Edit Distance
- Grid DP Problems

### Week 7: Advanced Topics
- Greedy Algorithms
- Divide and Conquer
- Backtracking
- Bit Manipulation

### Week 8: Final Review & Practice
- Mock Interviews
- Practice Tests
- Reinforcement & Optimization

## 🚀 Getting Started

### For Local Development

1. Clone the repository:
```bash
git clone https://github.com/ashishkhedekar02-blip/dsa-roadmap.git
cd dsa-roadmap
```

2. Open in VS Code and use Live Server:
   - Install the Live Server extension
   - Right-click on `index.html` → "Open with Live Server"

3. Or simply open `index.html` directly in your browser

### Direct Access
Open the `index.html` file in any modern web browser. No dependencies or installations needed!

## 💻 How to Use

1. **Browse the Roadmap**: View all topics organized by weeks and days
2. **Click Topics**: Click any topic tag to mark it as completed
3. **Track Progress**: See your progress count in the top right corner
4. **Filter by Week**: Use the dropdown to view specific weeks
5. **Access Resources**: Click the "Learn →" button to access tutorial videos
6. **Keyboard Shortcut**: Press `Ctrl+Shift+R` (or `Cmd+Shift+R` on Mac) to reset all progress

## 📊 Progress Tracking

- Your progress is automatically saved in your browser's local storage
- Completed topics appear with a green background and strikethrough text
- View total completed topics at the top of the page
- Progress persists even after closing the browser
